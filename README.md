# sieve-of-eratosthenes
The Sieve of Eratosthenes algorithm in Swift


## Usage

```bash
# Run the script and pass a number. 
# Note that you may need to run 'chmod +x ./main.swift'
./main.swift 10000000

# Result
[2, 3, 5, 7, 11, 13, 17, 19, 23, ...]
```
